# rubiks-cube

Rubik's cube interactive model.

Deployed at [rubik.jakubsmutny.cz](https://rubik.jakubsmutny.cz).

### Commands

Build the project with `npm run build`.

Run local development server with `npm run dev`.

Run tests with `npm run test`.
